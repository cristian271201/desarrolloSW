# desarrolloSW - TP_HTML
Instrucciones para ejecutar el archivo HTML.
El archivo HTML muestra una página web sencilla que contiene aspectos básicos sobre el manejo de encabezados, listas, elementos multimedia, tablas,etc.
El archivo es compatible con cualquier navegador, pero es importante contar con acceso a internet para poder visualizar los elementos multimedia.

PASOS para abrir el archivo HTML en un navegador:
1. Descargue el archivo "index.html" desde el mismo repositorio donde se encuentra este README
2. Localice el archivo en su computadora.
3. Haga doble click en el archivo "index.html" y se abrirá con su navegador predeterminado.

PASOS si desea ver el código del archivo en Visual Studio Code y tiene instalada la extensión Live Server
1.  Descargue el archivo "index.html" desde el mismo repositorio donde se encuentra este README.
2.  Localice la carpeta donde se encuentra el archivo en su computadora.
3.  Haga click derecho y luego click izquierdo en la opción "Abrir en terminal"
4.  Coloque el comando "code ." y presione enter.
5.  En la esquina inferior derecha aparecerá "Go Live". Al clickear allí le abrirá la pagina en un navegador predeterminado y podrá ver los cambios realizados en el código en tiempo real
